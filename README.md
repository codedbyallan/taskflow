# TaskFlow

Gerenciador inteligente de tarefas desenvolvido como projeto prático de portfólio.

## Objetivo

Criar uma aplicação web simples para cadastrar, listar, editar, concluir e excluir tarefas.

## Stack prevista

- Backend: .NET / ASP.NET Core Web API
- Frontend: React + Vite
- Banco de dados: MongoDB Atlas
- Deploy: Render
- Versionamento: Git + GitHub

## Status do projeto

Projeto iniciado. Estrutura inicial criada.
